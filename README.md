# physio_toolbox

This repo contains scripts for recording physiological data from MATLAB

requirements:
  * Measurement computing USB analog-to-digital device
  * MATLAB 32-bit version (MCC device not supported in 64-bit MATLAB)
  * data acquisition toolbox for MATLAB

features:
  * can record heartbeat, respiratory, GSR (untested) and scanner triggers
  * configurable sampling rate
  * integrates into MATLAB/Psychtoolbox scripts easily
  * also runs stand-alone
  * can start acquisition with keypress or off scanner trigger
